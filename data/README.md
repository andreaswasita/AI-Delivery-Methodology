# Data Platform Components

This directory contains data platform infrastructure components for the AI Delivery Methodology, specifically Microsoft Fabric resources and related data platform tooling.

## Contents

- **FABRIC-LANDING-ZONE.md** - Comprehensive guide for Microsoft Fabric landing zone deployment with compliance and security controls
- **fabric/** - Microsoft Fabric infrastructure components
  - **scripts/** - Automation scripts for Fabric deployment and management
  - **terraform/** - Terraform configurations for Fabric infrastructure

## Microsoft Fabric

Microsoft Fabric is a unified analytics platform that consolidates:
- Data Factory
- Synapse Data Engineering
- Data Science
- Real-Time Analytics
- Data Warehouse
- Power BI

All Fabric components in this directory are designed with compliance-first principles, supporting GDPR, SOC 2, ISO 27001, PCI-DSS, APRA CPS 234, HIPAA, and other industry-specific regulations.
