## How to Use This Checklist

This checklist guides you through all activities required to successfully mobilize and initiate an AI project following Microsoft's AI Frontier delivery methodology.

**Instructions:**
- Check off items as completed
- Add notes/dates in the comments column
- Flag any blockers or risks
- Review weekly with project team
- Use as input for phase gate review
