üéØ Deployment Options

Option 1: One-Click Portal Deployment (Recommended for Quick Start)

Click the "Deploy to Azure" button above. You'll be taken to Azure Portal where you can configure:

  • Environment (dev, test, uat, prod)
  • Azure region (e.g., Australia East, Southeast Asia)
  • Project name and organization details
  • Enable/disable compliance features

Option 2: PowerShell/CLI Deployment (Recommended for Automation)

For CI/CD pipelines and infrastructure as code workflows:

# Clone repository
git clone https://github.com/andreaswasita/AI-Delivery-Methodology.git
cd AI-Delivery-Methodology/infrastructure

# Login to Azure
az login
az account set --subscription "Your-Subscription-Name"

# Deploy using PowerShell script
./scripts/deploy.ps1 `
    -ResourceGroupName "rg-ai-platform-prod" `
    -Location "australiaeast" `
    -Environment "prod" `
    -EnableCompliance $true

Option 3: Bicep Direct Deployment

Use Azure Bicep CLI directly for maximum control:

az deployment sub create \
  --location australiaeast \
  --template-file infrastructure/bicep/main.bicep \
  --parameters infrastructure/bicep/parameters/prod.parameters.json
